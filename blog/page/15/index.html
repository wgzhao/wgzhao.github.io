创建了长沙Linux爱好者豆瓣小组

| Comments

很羡慕一些地方的Linux用户小组能有时间，有人定期的举行一些技术沙龙活动，相互交流，共通进步。
回长沙后，也一直想找到这样的小组或者创建这样的小组，而不让自己觉得独孤，但是一直没有想到一个好的办法。
后来看到豆瓣上有不少活跃的小组，感觉比较好，特别是看到了长沙豆瓣小组里有很多具体的分类，看上去还不错，因此就萌发了创建长沙Linux豆瓣小组的想法。

晚上收到了豆瓣的确认邮件，长沙Linux豆瓣小组创建成功。

So，如果你在长沙，你是Linux的爱好者，或者你打算了解Linux，可以尝试加入这个小组，至于外地的，不是我拒绝大家，而是觉得可能效果不太好，当然你愿意跑来跑去，欢迎你的加入。

长沙Linux爱好者豆瓣小组的地址是：
http://www.douban.com/group/changsha-linux

| Comments

考虑买这本书，暂时在网络书店还没有看到有销售了

下面是这本书的介绍：

这是一本涉及范围颇广的实战技巧汇编著作，涵盖 了作为 Linux网络管理员需要了解的所有内容，无论你是入行新手还是具有多年经验的资深人士。《Linux Networking Cookbook》介绍了完成一系列任务的完整步骤，可以帮助你深入直接地掌握搭建和维护计算机网络的实际操作。每一节都包含了明确的解决方案和测试代码，以及为什么如此与怎么样实现的相关讨论。.
可以运行一个网络并不能说明你就掌握了所有答案。
《Linux Networking Cookbook》描述了与网络服务相关的一些解决方案：防火墙、无线接入点、安全的远程管理、远程帮助台、用户远程访问、虚拟专用网（VPN）、验证、系统和网络监控，还有快速增长的 IP 语音（VoIP）服务等。

使用 OpenLDAP 集中网络目录 使用Nagios 或 MRTG 进行网络监控 了解和熟悉 IPv6 设置新系统的自动网络安装 通过串行控制台进行 Linux 系统管理
《Linux Networking Cookbook 》还包含了 Linux 和 Unix 服务器之间联网、整合 Windows主机、路由、用户识别和验证、共享 Internet 连接、连接分支机构、域名服务、有线和无线连接、安全、监控和故障诊断等任务解决方案。
当你需要快速解决某个网络问题，却没有时间或者耐心通过搜查参考书籍或者 Web 来寻求答案的话，那么这本书就是你所需要的。…

阿猫，走好！

| Comments

本以为我能写下更多的想念，下笔时才发现，脑中的思绪无法行程文字。

千言万语缩成一句话，阿猫，一路走好！

收到短信的时候，我乍一看以为是垃圾短信，但是发信人确实我的高中同学，再认真看内容，还是觉得不太相信，于是打个电话追问过去，问短信的内容是不是真实的，他说是的。

也许不太严重吧，于是我又问目前在哪个医院，回答是在殡仪馆，一个那遥远，冰冷的字眼，同时也让我的心冰冷了起来。

29岁，多么年轻，生活才刚刚开始，孩子2岁多了，早应该可以叫爸爸了，可从此以后，他该去哪里叫爸爸呢？

依然记得我们总是喜欢拿他的名字开玩笑，每次老师说要点一名女同学回答问题的时候，我们就会怂恿老是点他的名字，然后接着等他慢慢的站起来的时候，教室里一片欢笑声，他的脸就更红了。

依然还记得他有一段时间痴迷于书法，但是那个时候大家都不富裕，买不起好的钢笔，于是专卖钢笔尖的市场就出现了，每次他都会买几个，特细的，书法的，粗旷的，然而总是不能等到下次购买的时间，这些笔尖就已经磨秃了。

依然还记得每次打乒乓球时，他使用的各种招式，于他飘逸的头发很是能构成一副大侠的POSE形象，虽然那些招式的确不怎么实用。
