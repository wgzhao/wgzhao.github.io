Intel 版 Mac OS X 10.4.4 被破解

| Comments

Apple 最新的 Intel 版 Mac OS X 10.4.4 已经被破解高手 Maxxuss成功破解

Maxxuss 在他的网站上说道，”这是我对于官方版 Intel 平台 Mac OS X 的补丁方案的初步版本。最终你将可以从任何普通 x86 计算机上（需要支持 SSE2）运行这个 Mac OS X。还有许多工作和文档需要做，例如对于只支持 SSE2 的 CPU，需要有一个正式的安装过程和 PPF 补丁。然而，如果你想先玩玩的话，你就可以用现在这个版本开始了。“

Google Talk for Linux 不远了

| Comments

Google Talk 做为一个和gmail集成在一起的即时聊天工具，从一开始就引起了很大的关注，只是目前还只有windows版，这对我等Linux Fans 而言，无疑是一痛苦，不过刚看到下面的这个新闻，估计Google Talk 马上就有linux版本了。

据DesktopLinux报道，Google正与Codeweavers(Crossover Office)合作，开发起照片管理软件Picasa2的Linux版本。该版本将使用wine来运行Windows下的Picasa，而这对于 Linux用户是透明的，他们将可以像使用其他Linux软件一样方便的使用该版本的Picasa2。

正在开发的Linux下的软件还包括Google Talk等。

codeweavers的Crossover office我一直在使用，确实比简单wine功能要强大很多，我测试过photoshop 7.1,windows office 2003,dreamweaver MX,IE 6.0 SP1等，还包括一些其他的小型应用程序，基本上和在windows平台运行起来没有太多的区别，只是对中文的支持很糟糕。

期待着 Google Talk for Linux的出现！
