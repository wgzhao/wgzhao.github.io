三个月就讨论出这样的结果？

| Comments

南方周末 2007年8月16日第一千二百二十七期 A4版《中国药监系统“刮骨疗伤”》中有这样一段话

……2007年1月26日到3月30日，药监系统又开始了一场“监管思路大讨论”。 药监系统人士认为，这是一次非常难得的讨论，是继去年讨论的深化，重点是回答“为谁监管和怎样监管”的问题。讨论的结论是：应该把公众利益而不是商业利益放在第一位……

真没有想到，我们药监系统行业的精英们，仅仅只要花3个多月的时间，就能得出“应该把公众利益而不是商业利益放在第一位”的结论，真是来之不易呀。

这是否也就意味着，药监系统以前一直得到的最高结论是”应该把商业利益而不是公众利益放在第一位“呢？恐怕真的是这样，否则怎么那么多公众利益受到侵犯的时候，商家们却看上去一个比一个过得滋润呢？

原来，真理其实不是掌握在少数人手里的，歪理才是！

Add/change Some Options for Firefox

| Comments

1)exit your firefox 2)open or create a file named user.js,and include the following lines
user_pref(“config.trim_on_minimize”,true); user_pref(“browser.sessionhistroy.max_total_viewers”,8); user_pref(“browser.cache.memory.capacity”,16530); user_pref(“browser.cache.memory.enable”,true); user_pref(“browser.urlbar.clickSelectAll”,true); user_pref(“browser.urlbar.hideGoButton”,true); user_pref(“dom.pop_maximum”,5); user_pref(“extensions.dss.enabled”,true); user_pref(“network.dnsCacheEntries”,40); user_pref(“network.dnsCacheExpiration”,1200); user_pref(“network.dns.disableIPv6”,true); save and exit. 3) put the file to the path including your firefox working path.i.e ~/.mozilla/firefox/q04xpu0l.default. 4)run your firefox again.
