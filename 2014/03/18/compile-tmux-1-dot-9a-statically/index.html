静态编译tmux 1.9a版本

Mar 18th, 2014 | Comments

tmux 的最新版本 1.9 带来了很多特性，而 CentOS 的仓库里，目前我只能找到1.6的版本。因此想要自己编译一个，但是tmux 1.9a编译时要求 libevent 为2.x的版本，而 CentOS 6.x 默认还是1.x的版本，为了不影响系统的稳定性，决定采取静态编译的方式。流程如下:

  1. 系统首先安装需要的几个静态库

     sudo yum -y install glibc-static ncurses-static

  2. 下载最新的 tmux 1.9a

  3. 下载 libevent 2.0 源代码

  4. 静态编译libevent

     tar -vxzf libevent-2.0.21-stable.tar.gz
 cd libevent-2.0.21-stable
 ./configure --enable-static --disable-shared
 make && sudo make install

  5. 静态编译tmux 1.9a

     tar -vxzf tmux-1.9a.tar.gz
 cd tmux-1.9a
 ./configure --enable-static --prefix=/usr/local
 make && sudo make install

如果上述步骤觉得比较啰嗦，可以点击这里下载我已经编译好的版本。

Update 2014-03-22

目前 Mac 平台的 Macports 还没有提供1.9a的版本，我顺便编译了一个 Mac 平台的。可以从这里 下载。注意需要libevent 2.0版本。
