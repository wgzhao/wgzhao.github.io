Redhat El5如何映射裸设备到逻辑卷

| Comments

之前的版本如何做，这里不做描述，自行 Google。之前通常的方法，在该版本，KERNEL 并不能直接获得设备名称或者主次设备号，我们需要通过dmsetup指令来间接获得。一下操作的目录均在/etc/udev/rules.d/。

  1. 编辑 90-dm.rules 文件，注释掉下面这行
    ﻿KERNEL=="dm-[0-9]*", ACTION=="add", OPTIONS+="ignore_device"

  2. 编辑60-raw.rules文件，加入下面几行：

     KERNEL!="dm-[0-9]*",GOTO="skip_dm"
 PROGRAM!="/sbin/dmsetup  ls --exec /bin/basename -j %M -m %m", GOTO="skip_dm"
 RESULT=="vg1-lv1", RUN+="/bin/sh -c '/bin/raw /dev/raw/raw1 $MAJOR $MINOR'"
 LABEL=skip_dm

    上面第一行代码表示如果不是 dm 设备，则直接跳到 skip_dm 处第二行表示如果后面的指令返回为假（非 0 值），则跳到 skip_dm 处。
