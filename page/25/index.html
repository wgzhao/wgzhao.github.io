wirlfly和他的apue感想

最近和wirlfly交流得比较多，除了技术方面的，还有一些方向方面的讨论。

和他在技术方面的讨论时，出现的最多的一个缩写，恐怕是 aupe 了，他第一次写出 apue 的时候，并没有告诉我这是什么，但是我知道是书的时候，我立刻知道他说的是那本超级经典书了－－《Advanced
Programming in the Unix
Environment》， 用他的话说，这本书带他“进入了代码的美妙世界”，是呀一本好书读起来是如饮甘醇的，甚至还能影响一个人的一些思想。

TAOUP－－《The Art of Unix Programming》我不是一口气看的，是看一段，休息，再看一段，因此到现在还没有看完，虽然在感叹 ESR 也是奋青的同时，不得不感叹他对 UNIX 社区一些一针见血的看法这些看法无疑会影响我对 UNIX 和 Linux 的看法。
好书还是应该细细去读的，apue 虽然在一年前看过，而《TCPIP 详解》三卷也已经摆在我的案头上已经很旧了。

今天神奇般得见到了LIUDIEYU，也看到了他最新得很酷的个人主页以及行如流水般的英文，又一次刺激我英语也要看了。

是呀，要看的很多呀，而手头的杂事似乎很多，开发的效率也每况愈下，不知道是脑力的问题，还是因为重复做一件事情就会这样。

无论如何，该学习了！

申请google adsense不容易

最早申请Google
adsense是在 2005 年的 11 月，当时给出的拒绝是资料填写不完整或者违反相关政策，我想决绝的理由应该是后者了，应该是访问量达不到吧。
后来访问了多了一些的时候，我又一次申请 Google Adsense，时间是 2006 年的 1 月底，这次没有拒绝我，只是给了一个石沉大海的回音。

两次打击后，几乎没有再次申请 Google Adsense 的欲望了。但是……

上次brisy来我这里，要我帮他申请了 adsense，我向他描述了我申请的惨痛经历，也随便填写了一些资料。
4 天后，brisy 告诉我他的申请通过了，而且目前已经有收入了。这对我算是不小的刺激吧。他的访问量没有我高，但是 PR 值就比我高多了，但是我也看到很多没有访问量也没有 PR 值的 blog，怎么能申请到 adsense 呢？

还是再申请一次吧，这次不填写我的资料，填写lancy，她有固定地址，固定电话等，我现在是什么都没有，填写她的应该靠得住一些。事后证明这个决策是英明的，4 天后，我的 google 申请也通过了，折腾了半天在我的 blog 上和论坛上都投放了一些广告（大家不要给我面子，只管点击，呵呵）。效果嘛，目前还没有瞧见。用 brisy 的话说是“觉得心里踏实一些”。我到没有这个想法，管他呢，先投着吧。

居然如此不信任Linux

现在很多电脑厂商为了降低成本，很多机型在出厂的时候，采用了 Linux 系统，而不是 Windows 系统，其实这对 Linux 的普及无疑是一个很好的机会，但是如果你遇到这样的厂商客服，估计就不会想这是 Linux 的机会。下面是一个 Linux 爱好者因为出厂自带的光盘读盘有点问题而打进的电话，摘录一段：

话务员：您真的是要安装 linux 嘛？
用户：是的
话务员：您使用电脑做什么呢？
用户：上网、办公、聊天什么的………
话务员：您要上网、要使用 office 等是嘛？
用户：是的
话务员：如果您真的是要用上面用途，建议您不要安装 linux，因为 linux 无法实现
上面的功能，不兼容 widnows 软件
用户：那么你们为什么要集成呢？我买电脑的时候就带了这个系统啊，为什么不能用？
话务员：这么跟您说吧，因为国家的政策规定在 PC 出场的时候要代一个操作系统，
所以才带红旗的
用户：真的不能用嘛？
话务员：建议您还是安装 XP 吧，要是用 linux，你周围的人基本上都不会用的
用户：如果我一定要安装呢?
话务员：如果您一定要安装 linux，可以给您更换光盘，您可以打维修的电话
xxxxxxxxx；如果您没有特殊用途，还是安装 windowsXP 吧
用户：什么特殊用途可以安装 linux 呢？
话务员：反正就是你的用途不能满足吧………红旗的 800 你知道吧，可以问一下他们
用户：知道，谢谢
