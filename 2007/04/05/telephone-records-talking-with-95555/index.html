与招行95555的电话记录

| Comments

莫名其妙中，我的招商银行网上银行专业版不能使用了。于是给他们的打电话前面一通乱七八糟的提示过后，一位男士接了电话，很有礼貌的问我需要什么帮助，下面便是我们对话的记录： 

 我：我使用了你们的网上银行专业版，今天我做了一次证书恢复，提示是成功的，但是我无无法登录，输入密码后，直接给出一个对话框：无效用户#21.
 客服：请问您是用笔记本吗？我：是的客服：请问您用了软键盘吗？我：没有。客服：请稍等
 ……….
 客服：不好意思，久等了，您把出错的信息完整的报出来吗？我：输入密码后，点击登录，弹出一个对话框，上面就是一句话，无效帐号#21。客服：帐号那一栏有您的中文姓名吗？我：有客服：请稍等。
 ……………….
 客服：不好意思，久等了。请问您使用的是平板电脑吗？我:不是。客服：请稍等。
 …………………..
 客服：不少意思，久等了，只有这个报错吗？还有别的吗？我：没有，输入密码后，点击登录，直接报错，应该根本就没有去链接网络。客服：请稍等。
 …………………..
 客服：抱歉，久等了。要不您这样，您把错误的截图发给我们，我们要后台的技术人员帮您核实一下。您完成截图后，请发到下面的这个这个邮箱，我们技术人员会帮您处理的。我：那好吧，等下我吧。我：你们招行的网上银行 web 版的，除了支持 IE 外，还支持别的浏览器吗？客服：是这样的，我们有客户反映后，有时能支持，有时不能支持。我：什么叫有时能支持，有时不能支持呀？客服：请稍等
 ………………………
 客服：不好意思，久等了，就是和其他浏览器有兼容性问题，我们建议您使用 IE 浏览器。我：但问题是，我没有 IE 浏览器呀，我也不想使用他呀。客服：您可以下载 IE 浏览器来安装。我：我没有安装呀，我用的是 Linux 操作系统。客服：请稍等
 ………………….
 客服：您说的是 Linux 系统吗？很抱歉，我们这套系统是基于 windows 平台开发的，所以对 Linux 并不支持。我：那我这种情况下怎么办呢？你们有什么解决办法呢？客服：很抱歉，现在没有办法解决。我：那什么解决，我就想知道，我什么时候可以在我的系统上能用上呢？客服：很抱歉，我们这里是用户故障处理中心，后台开发我们也不并清楚，所以我们也不知道什么时候能支持。我：那意思是现在没有办法支持罗？客服：对，现在就是不支持。
 ……………………………
 我知道，我再说下去，也没有什么含义。

我之所以问网上银行对浏览器的支持问题，一来我是真的想问问，他们的那群研发从哪个角度考虑的，采用了这种毫无可迁移性的 ActiveX 方式； 二来，大家支持，徐继哲写一个一篇致招商银行的公开信，也是声讨他们的网上银行的问题，得到了很多人的响应，很多媒体也报道了，连国外的媒体也报道了。我这也是一种声援了。
