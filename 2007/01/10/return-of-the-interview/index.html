面试归来--明天就上班？

| Comments

我猜中了开始和过程，但是没有猜中结尾！

14：33，到了中国同学录的工作地点—电信大厦 27 楼。因为没有前台，因此拿出手机给昨天联系我的人打电话，一看手机，没有信号。果然是人家的地盘呀，在高速电梯上，小灵通信号覆盖率是 100%，而在这里，移动信号覆盖率几乎成了 0%。

找了一个空旷的地方，总算有了一个信号，信号还真的不好，断了两次才算说完“我已经到了 27 楼，怎么找您”这句话。

他们下午 14:30 上班，不过现在还没有到，那就等吧，随便看看它们走廊上的宣传画。面对电梯出口的墙壁上是大幅的中国同学录宣传画。其左侧是胡哥提出的”八荣八耻“的宣传，这应该是所有事业单位的标识吧。对面墙壁有两幅，一副是中国电信的口号—”为搏而生存“，另外一个是 114 号码百事通的宣传广告。

等我把这四副宣传画看到几乎能做成找别扭的程度后，总算等待的人来了，现在是 15:03。事业单位上班时间果然是弹性制呀！

其实在边等待的过程中，我一直在思考我应该如何回答它们准备提出的问题，因为我他们一定会问一个问题，那就是 java 和 jsp，因为中国同学录目前全部是 jsp 界面的。

果然，坐下来就是就是这个问题，于是我说我会去学，应该会比较快。剩下的问题就比较简单了，问我对 Linux 了解的程度如何？对 Linux 下的服务配置了解如何？ 比如 DNS 服务，web 服务？问我对一般用什么 web 服务器，我说 apache 和 zeus，于是就 zeus 的介绍，花了我不少时间。然后指着我简历上写的了解 NAS,SAN,iSCSI 问我，你了解多少？于是又花了不少时间来解释 iSCSI 的问题。

接下来的事情，便是我能猜得到的，大致他们的话是这样的“我觉得你做系统维护更好，我看你的能力做我们现在的维护应该是绰绰有余，现在我们有两个做系统维护的，能力不怎么样，你怎么不面试系统维护呢？”

于是我又花了不少时间来告诉他们我不想把我的重心放到系统维护上，想转型。而实际上，它们仅仅是一些 linux 服务器，然后就是 oracle 数据库，这样的系统维护，我觉得没有必要花去我的全部精力，因此我拒绝了。

得到这样的答复后，它们就开始讨论起我上班要做什么事，什么时候来上班的问题，这就是我始料不及的了。聊完后，于是它们要求我在一个月的时间内，JAVA 技术必须从现在的零升级到独当一面。在我答应后，他们加了一句，以后系统维护有什么难题，要是要你去帮忙。这是不是有点交完钱付了帐，突然说，你看这个你送一个算了吧的情形。

接下来便是说它们的薪资体系。介绍完后，一个老总便说你等我们的通知吧。另外一个老总就说你过来填一张表吧，跑到一个文秘那里，文秘却不知道要填什么表，只是问，是不是要了，这个老总说，嗯。

这有点意思，刚才不是说要等通知吗？

文秘问我，你什么时候能来上班？我说随时。“那你明天来吧！”没有想到这么干脆，呵呵。

但是等这个老总带我去和刚才老总说一声的时候，它们就发生了意见分歧，一个说明天来上班，一个说下周一吧。讨论了半天，最后说，你明天来吧，到时我看能不能找一个位置给你。

到此面试结束，看时间，15：42！

我真的明天要去上班？然后真的开始我的新的方向—JAVA？

一直以来，我都没有碰 JAVA，因为我一直以来都是希望自己朝技术支持，系统集成这个方向发展，因此仅仅是去了解了 C 语言。而 web 开发的最初目的并不是我要去学习 web 开发技术，而是因为如果我学了 web 开发，我就能迁移学院的网站，这样我就有了一台对我的服务器，而这一直以来是我希望得到的，于是便这样断断续续做到现在。

现在突然要转型真的做开发，我还真没有做好太多的准备，而且一个月之内,JAVA 编程技术由零变成独挡一面，我不知道结果会如何？
