用好sudo

sudo 工具能够帮你把一部分系统管理的责任分摊给其他人，还不用给出完全的 root 访问权限。它是一个 setuid 程序，只需要输入用户自己的密码，就可以执行 sudo 后面的命令。
使用 root 帐号，运行 /usr/bin/visudo 来编辑那些用户可以调用 sudo 命令的用户列表，缺省的 sudo 列表看上去类似下面这样：

 root ALL=(ALL) ALL
