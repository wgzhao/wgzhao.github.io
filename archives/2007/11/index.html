买了第一只股票

开户已经有 2 个星期了，一直不方便，所以也就没有买。
上周好玩，想买进工商银行 (601398)，因为定的买入价格比较低，一直都没有买进。
今天上午，首先定价 8.00 买进，发现不太可能，于是改为 8.10，发现还是买不进去，看看信息，有大单在 8.15 处买进，于是我也随波逐流，设在 8.15 买进。
果然买进了，不过接下来就是跌呀跌跌呀，当时设定的 8.10 早就可以买进了，于是我来设定 8.00 买进，管它呢，买进了就买进了，没有就算了。
这算是我买的第一只股票了。

然后顺便打算在 10.60 处买进建设银行 (601399), 到目前为止，还没有买进。

这纯属于试水，了解一些基本行情和投资的基本套路，所以一共只搞 2k 来玩，呵呵。

11月 6 2007

PhysicalDrive0

在 Linux 里使用 qemu，当需要从本地实际硬盘启动和运行 OS 的时候，只要给出-hda /dev/hda这样的参数就可以了。
因此 Linux 要表示某一个硬盘，实在是很简单，虽然很多 Linux 初学者觉得 /dev/hda 这样的名字怪怪的。
而在 Windows 下，想表示某一个硬盘，可不是那么简单，表示某一个分区也没有那么容易。我最开始在 Windows 下使用 qemu 要运行我本地硬盘的 linux 的时候，我就找了很久的资料来确定如何表示一个硬盘。
如果做过 windows 驱动程序开发的人，估计就不用我这么累的去找了。
今天又要用 qemu 临时启动我的 DC5.0，结果又把这个参数忘记了，费了半天劲回忆起来了，于是决定写到这里。也就是我标题写的了。
\\.\PhysicalDrive0表示第一个硬盘，以此类推，\\.\PhysicalDrive1是第二个硬盘。 不过不知道 windows 下是否区分 IDE 硬盘和 SCSI/SATA/SAS 硬盘的区别，应该不需要吧。哎，还是对 Windows 了解太少了。
再一次诅咒我的 CDMA，害死人呀！
