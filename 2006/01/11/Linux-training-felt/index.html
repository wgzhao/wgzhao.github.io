Linux培训有感

| Comments

这两天的培训又让让我走上了阔别 8 个月的讲台，再一次听到别人称呼我为老师。依然发现自己上课还是那样冲动，特别是讲授我的最爱的时候，这种情况就更加突出，每每激动不已。只是相比 1 年半前在讲台上讲述 Linux 时，发现自己对 Linux 的认识更加理性了一些，而不是再是在学校时的那种一味狂热，而不清楚市场，也不理解竞争的残酷和充分考虑用户的习惯。

我很清楚的认识到，想让 Linux 如同 Windows 一样走向千家万户，除了习惯以外，常用软件是很大的障碍。首先看看办公软件吧，我的机器上同时安装了 WPS2005 For Linux，RedOffice3.0,OpenOffice.20,结果谁也不能完全兼容谁，即使是在 OpenOffice 上做修改而成的 RedOffice 也不能 100%兼容它。这不得不使你在编辑你的文档时充分的考虑这篇文档你将会给哪些人，他们将使用什么样的办公套件。其实我个人是很讨厌使用 word 这种二进制格式的文档的，txt 就不错了，是在不行来一个 html，或者 tex 转换成 pdf 也行，因为至少不会由于丢失几个字节而导致我的文档没有办法打开。

其次便是网上冲浪，女友对此很有感触，在我的极力怂恿下，女友放弃了使用 Windows，而是学习我使用 Linux，刚开始给她安装的是 FC2，结果发现那需要掌握太多的命令，对中文的支持也不太好。

直到后来使用了红旗的桌面系统，才坚持下来。可是没当她需要浏览网页的时候，就是她最痛苦的时候，现在忙于找工作的她，最近总是需要在一些所谓的招聘网站上填写资料，每每花了很多时间填写的数据等你点击提交的时候，却没有任何反应，或者直接给出服务器错误，让你没有任何后悔的余地，往往 1 个小时可以搞定的事情，她需要花上一个晚上的时间来完成。

虽然我一再告诉她，其实这真的不是 Linux 的错，也不是 firefox 的错，而是那些该死的网站写的代码不遵守 W3C 标准的错，但是对于一个需要解决问题的用户而言，再多的技术解释也是无意的，解决问题才是硬道理，没有办法，只能在 Linux 系统上活生生的再整上一个 IE，是在有点别扭。剩下来的是娱乐，音乐和电影播放，Linux 上的软件已经做得很不错了，基本上都是万能播放器，mplayer 是我的最爱。他体积下，占用资源少，播放流畅，而且还能做视频音频转换，真是功能强大，性能极佳。只是游戏方面对于 Linux 而言却是一个永远的疼，几乎没有大的游戏厂商愿意开发 Linux 平台的客户端，市场决定了他们不会做这个投入。而有一些的就是不是很知名的厂商，我所知道的有蓝迪游戏大厅有几乎 Linux 平台的，只是人气还不是很旺。

如果 Linux 能解决上面的三个问题，我想离 Linux 的普及也就不远了。

到那个时候也许我们经常听到的一句话就是“hi，你电脑装的什么系统？redhat,suse，还是 redflag?“.
