X-windows启动特别慢是bug吗？

| Comments

同事遇到一个问题,启动KDE的时候特别慢，4，5分钟是常事，10多分种也出现过。而且做什么都慢。机器是IBM T43（羡慕ing…..)，配置应该不存在问题，之前也出现过，也不知道怎么折腾的，有时候就好了。一直以为是系统的bug。
今天问到我，恰好，我之前也遇到过x-windows启动特别慢的情况。于是很很快的在他的机器上执行
`ping hostname“`
果然ping不通，那很显然和我的遇到的情况一样了，熟悉X11的都知道，x-windows的client与sever之间通讯是采用TCP/IP的。主机名ping不通，那么在启动KDE和其他应用程序势必会长时间的等待。因此出现了上面的那个现象。
