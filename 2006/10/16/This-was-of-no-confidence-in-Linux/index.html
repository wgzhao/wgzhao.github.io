居然如此不信任Linux

| Comments

现在很多电脑厂商为了降低成本，很多机型在出厂的时候，采用了 Linux 系统，而不是 Windows 系统，其实这对 Linux 的普及无疑是一个很好的机会，但是如果你遇到这样的厂商客服，估计就不会想这是 Linux 的机会。下面是一个 Linux 爱好者因为出厂自带的光盘读盘有点问题而打进的电话，摘录一段：

话务员：您真的是要安装 linux 嘛？用户：是的话务员：您使用电脑做什么呢？用户：上网、办公、聊天什么的……… 话务员：您要上网、要使用 office 等是嘛？用户：是的话务员：如果您真的是要用上面用途，建议您不要安装 linux，因为 linux 无法实现上面的功能，不兼容 widnows 软件用户：那么你们为什么要集成呢？我买电脑的时候就带了这个系统啊，为什么不能用？话务员：这么跟您说吧，因为国家的政策规定在 PC 出场的时候要代一个操作系统，所以才带红旗的用户：真的不能用嘛？话务员：建议您还是安装 XP 吧，要是用 linux，你周围的人基本上都不会用的用户：如果我一定要安装呢? 话务员：如果您一定要安装 linux，可以给您更换光盘，您可以打维修的电话 xxxxxxxxx；如果您没有特殊用途，还是安装 windowsXP 吧用户：什么特殊用途可以安装 linux 呢？话务员：反正就是你的用途不能满足吧………红旗的 800 你知道吧，可以问一下他们用户：知道，谢谢
