Xplore何去何从

| Comments

很多人问我 Xplore 是什么意思，他和 linux 怎么会扯上关系。 Xplore 并不是你在 google 搜到的 Xplore 品牌手机，当然也不是 xp 知识库了（xplore == xp lore)。 xplore 就是 Xplore，它只是我们为自己的工作室取的名字，它是取单词 explorer 的一部分，很显然，这是很多公司或者品牌取名的办法，而且我喜欢 X 开头的名字。

但是工作室主要成员是 brisy,dieyu,glemir,mlsx,xinhe 和 zwell。我们工作室的中文名字是伊克斯托，这同样按照惯例采用音译的方式来取的，是由 lancy 帮我们想到的。成立的时候只是想我们各自做的一些作品展示一下，同时整合大家的资源，能够做一些项目。不过成立后并没有达到我们的目的，在湘潭这样的地方，你不太可能找到太多的 IT 项目，于是我们接二连三的只是做一些网站，学校的，个人的，一些公司的。brisy 则负责全部的美工。当时，我正在教授 Linux 课程，我非常渴望能在学校推广 Linux，于是冒着学校不允许私自 [color=#FF0000]假架设论坛的风险，我开了 xplore 论坛，主要内容是 Linux 方面的一些东西，也就是大家现在能看到的。用户的习惯是可怕的，学校的教育模式是恐怖的，在成为微软产品培训基地的学校，想要大家使用 Linux 何其容易，虽然当时有很多人感兴趣，但是那仅仅是感兴趣而已，也许会下载光盘安装一些，接下来的事情可能就是如格式化 Linux。加上当时的成员面临着毕业，找工作的难题，论坛一直处于半维护状态，加上空间一直不怎么稳定，首先是放在学院的服务器上，外面的人访问如同 56kmodem 拨号一般，遇到特殊情况，还封锁网络。后来找了一个广西的网吧老板，把论坛从 linux 迁移到了 windows，这也许是一个莫大的讽刺，一个专门讨论 Linux 的论坛却假设在 windows 服务器上。网吧毕竟是网吧，结果论坛第一次厄运降临了，连续一个月不能访问，包括我们。找不到服务器的维护人。其结果可想而知，等我们挽救回来时，大部分人以为它和其他论坛一样，悄然消失了。从那时起，决定寻找更好的空间。后来我们找到了沅陵一中的老师，我们安装系统后，就落脚在这里了，一直到现在。期间也是一波三折，不过还好，没有出现过长时间不能访问，不过类似今天下午因为停电不能访问的日子倒是很多，特别是到电力紧张的时候。一直在找一些好的办法来提升论坛人气，期间我们也尝试了多种方法，但是都没有太多起色，团队人员少是一个很重要的原因，大家都有自己的工作，都很忙，没有太多的时间来维护它。速度和可连续访问也是一个问题。 Xplore 该怎么办？ uplinux 是一个非常优秀的 linux 专业网站和社区，主页有相当高的名气，论坛注册人数也不少，uplinux 的站长恰好是我同事，我们讨论了合作的可能性，很显然，这是一次愉快的探讨，我们达成了一致，Xplore 将和 uplinux 进行全们整合，并发展其首页，我们首先要做的是两个论坛的整合，首页的整合。接下来我们会把 uplinux 的首页朝门户网站方向接近，力争做最好的 linux 技术服务网站。而 Xplore 仍然会是工作室的名字，它是 uplinux 站的技术支持中心，并且在这个工作室的努力下，我们会推出其他的产品，而不仅仅是 uplinux。也许就是下周，大家可以看到一个不一样的 Xplore 和 uplinux。
