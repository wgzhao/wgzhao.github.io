北京全球通数据库搭建在什么系统上？

| Comments

连续四天没有交上电话费，去银行不行，自助缴费也不行。准备去营业厅缴费，结果银行的服务人员告诉我，去那里也交不了，“为什么”，我问。

“他们打电话给我们，说系统感染病毒了，正在处理！”。

病毒感染？难道他们使用的是windows系统？那我的欠费是不是会神奇的变成缴费过多呀。似乎可以把这个消息告诉销售，让他们使用Linux系统好了。

等待报销的日子又长了。
