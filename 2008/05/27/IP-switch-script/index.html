IP切换器脚本

| Comments

不同的网络环境下，需要配置不同的IP地址，或者应用不同的IP配置方式。每次配置时，都是修改网卡配置文件，比较麻烦。比较理想的做法是记住每次的配置文件，下次需要时，只要应用找个配置就好了。

everest里有cjacker写的evlinks。图形化的配置界面，集成了有线网卡和无线网卡的配置，异常方便。

可惜目前只能在Everest上能用，我曾尝试迁移到debian系统上，没有成功，失败的大部分原因还是在于我对程序的不了解。

到网络上寻找了类似IP切换工具或者说多IP管理程序，都没有找到，Windows下倒是比较多。Linux几乎没有。于是产生了自己写一个脚本的方式。自己能用就好了。

今天花了几个小时，完成了这个脚本，在Asianux 3.0，kubuntu 8.04上测试通过。SuSe我没有测试。

我自己用还是足够了，但是别人用，我还不知道会不会出很多问题，程序写的很简单，很多需要校验的地方也没有校验。 慢慢的完善吧。

欢迎使用改脚本的人给我反馈信息，特别是SuSe系统的。
